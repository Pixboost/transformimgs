Usage Examples

In this quickstart, we setup an Image API using reverse proxy. All requests that start with /cdn will be forwarded to Image API that will fetch the original image from http://webserver/img/ URL and applies different resizing and optimising techniques.

Original vs Optimised Image

      
<img
  src="/cdn/http://webserver/img/parrot.jpg/optimise"
  alt="red parrot" width="2072" height="2590"/>

Responsive Image with High DPI support

In our opinion this is the best way to serve images in 2022.

We pass DPI hint to the Image API, so we can save some bytes when serving for high DPI displays.
